# Agent Operating Guide

This project uses the Hector Antigravity Universal Starter Pack.

## Default workflow
1. Use `project-context-digest` before starting substantive work.
2. Use `task-planner-harness` for multi-step changes.
3. Use `safe-command-harness` before terminal commands.
4. Use `privacy-secret-guard` before reading configs, logs, `.env`, credentials, or sensitive data.
5. Use `code-quality-harness` after code changes.
6. Use `git-workflow-harness` before commits or branch operations.
7. Use `docs-status-handoff` after each milestone.

## Safety rules
- Stay inside the project workspace unless explicitly asked.
- Do not run destructive commands without explicit confirmation.
- Do not expose or store secrets.
- Do not overwrite original datasets.
- Do not push to remote Git without explicit confirmation.
- Do not claim tests passed unless they actually ran.
